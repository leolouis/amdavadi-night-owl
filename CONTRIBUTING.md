# Contributing to Amdavadi Night Owl

We love that you want to help Ahmedabad's night owls find their way!

### How to add a new spot
1. Fork this repository.
2. Open `data/places.json`.
3. Add your spot following this exact format:
   ```json
   {
     "name": "Name of Place",
     "lat": 23.0225,
     "lng": 72.5714,
     "category": "Tea/Food",
     "description": "Short description"
   }
